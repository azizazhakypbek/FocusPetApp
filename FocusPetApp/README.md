
# FocusPetApp — Windows WPF Virtual Pet

## Requirements
- Windows 10 or 11
- Visual Studio 2022+
- .NET 8.0 SDK

## Build
1. Open FocusPetApp.sln
2. Press F5 or Build → Build Solution
3. The executable will appear in:
   /FocusPetApp/bin/Debug/net8.0-windows/

## Notes
- Place all decoded PNGs inside /Assets/
- The overlay window is draggable and always-on-top.
- Timer is fixed to Option A MVP.
